Rekonsiliasi & Akurasi Data

Software Utama Microsoft Excel
Objek Data SP2D Posted & Netto

Proses Rekonsiliasi adalah tahap krusial untuk memastikan bahwa setiap transaksi yang telah diproses oleh Teller sinkron dengan catatan administrasi di BPKAD.

Saya bertanggung jawab menginput data dari berkas berstatus "Posted" ke dalam format Excel yang telah disediakan. Hal ini meliputi pencatatan nomor SP2D, nilai nominal Netto, hingga identitas User Teller terkait untuk memastikan saldo harian tetap seimbang atau balancing.

Detail Teknis Pekerjaan:

  • Memisahkan lapisan berkas antara arsip Bank dan arsip BPKAD pasca proses posting.
  • Melakukan input data harian dengan tingkat ketelitian tinggi untuk menghindari selisih angka.
  • Melakukan rekapitulasi terhadap performa User Teller (seperti User 11404 dan 20750) secara berkala.
  • Mencetak laporan hasil rekonsiliasi untuk divalidasi oleh pembimbing lapangan di Bank Riau Kepri Syariah.
  • Memastikan database transaksi akurat sebagai bahan laporan pertanggungjawaban keuangan daerah.
